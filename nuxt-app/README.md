# nuxt-app

## Build Setup

```bash
# install dependencies
$ npm install

# serve with hot reload at localhost:3000
$ npm run dev

# build for production and launch server
$ npm run build
$ npm run start

# generate static project
$ npm run generate
```

```bash
# To get started:

$ cd nuxt-app
$ npm run dev

# To build & start for production:

$ cd nuxt-app
$ npm run build
$ npm run start

#For TypeScript users.
```
